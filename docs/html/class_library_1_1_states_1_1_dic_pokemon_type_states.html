Diccionario estático de estados donde cada CLAVE representa el nombre del tipo de un pokemon y cada VALOR es un EnumState. Dependiendo del tipo de pokemon se le asigna un estado por defecto.

Se utiliza en PokemonAdapter para asignarle un efecto de estado al movimiento especial.
