async Task< Pokemon > Library.Adapters.PokemonAdapter.GetPokemonAsync ( string pokemonName)
inline

Obtiene los datos de un pokemon por su nombre utilizando PokeApiService y los adapta a un objeto Pokemon.

Parámetros
pokemonNameNombre del Pokémon a buscar en la API.
Devuelve
Una instancia de Pokemon con los datos obtenidos, null en caso de error o si el pokemon no tiene al menos cuatro Move.
Excepciones
HttpRequestExceptionSe lanza si hay problemas al hacer la solicitud HTTP a la PokeAPI.

Este método usa propiedades JSON para extraer información sobre los datos necesarios para crear un objeto Pokemon.

Los Move son agregados al pokemon solo si tienen un power y accuary mayor a cero.
