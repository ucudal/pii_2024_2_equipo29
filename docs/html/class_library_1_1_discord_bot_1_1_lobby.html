Detailed Description

Representa un lobby para gestionar jugadores en espera y salas de juego. Implementa el patrón Singleton. Esto es util para tener una única instancia de la clase Lobby en todo el programa, por lo que nos permite evitar problemas en el emparejamiento de jugadores.

◆ TryToStartGame()

async Task Library.DiscordBot.Lobby.TryToStartGame ( InteractionContext context)
inline

Intenta iniciar una partida si hay suficientes jugadores en la lista de espera.

Parameters
contextEl contexto de la interacción de Discord.

Para iniciar la partida. Se necesita que la cantidad de jugadores esperando en el lobby sea igual a la cantidad máxima de jugadores permitida en la clase Game. Por defecto, 2 jugadores.

Si la partida es iniciada se crea una GameRoom y agrega a los jugadores emparejados a la misma. Además, se vacía la lista waitingPlayers para que otros jugadores puedan buscar partida.
