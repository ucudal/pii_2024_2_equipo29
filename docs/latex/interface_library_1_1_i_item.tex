\doxysubsection{Detailed Description}
Interfaz que representa un ítem que puede ser utilizado en un Pokémon. 

Esta interfaz define las propiedades y métodos que deben implementar todos los ítems que se pueden usar en el contexto de un juego de Pokémon.
