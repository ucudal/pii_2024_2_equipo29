El juego cambiará de turno automáticamente si ocurre una de las siguientes acciones\+: 
\begin{DoxyItemize}
\item Se ejecuta el comando {\ttfamily /\+Change} de forma exitosa.  
\item Se ejecuta el comando {\ttfamily /\+Attack} de forma exitosa.  
\item Se ejecuta el comando {\ttfamily /\+Item} de forma exitosa.  
\end{DoxyItemize}

Este comando fue creado a causa del siguiente problema, cuando un pokemon de un jugador se encuentra afectado por {\ttfamily Sleep}, se quedó sin pociones y a su vez es el único pokemon que tiene disponible el juego no cambiará el turno de forma automática, porque no se cumplen ningunas de las 3 condiciones presentadas en la lista de forma exitosa. 

Requisitos para la ejecución exitosa del comando\+:


\begin{DoxyItemize}
\item El usuario debe estar en un canal de batalla. De lo contrario, se envía un mensaje de error indicando que el usuario debe estar en un canal de batalla.  
\item El juego debe estar en curso para que el turno pueda ser cambiado. Si la partida no está iniciada, no se podrá realizar la acción.  
\item Si el comando se ejecuta correctamente, se cambia el turno y se muestra un mensaje de confirmación.
