\doxysubsection{Descripción detallada}
Clase que representa una Super Poción, un ítem que puede curar a un pokemon. 

Esta clase implementa la interfaz \doxylink{interface_library_1_1_i_item}{IItem} y permite curar a un pokemon restaurando una cantidad específica de puntos de salud (HP).
