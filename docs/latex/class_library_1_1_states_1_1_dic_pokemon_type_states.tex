Diccionario estático de estados donde cada {\itshape {\bfseries{CLAVE}}} representa el nombre del tipo de un pokemon y cada {\itshape {\bfseries{VALOR}}} es un {\ttfamily Enum\+State}. Dependiendo del tipo de pokemon se le asigna un estado por defecto. 

Se utiliza en {\ttfamily Pokemon\+Adapter} para asignarle un efecto de estado al movimiento especial.
