\doxysubsection{Detailed Description}
Clase que representa a un jugador en el juego, que puede manejar un equipo de pokemon. 

Esta clase implementa la interfaz \doxylink{interface_library_1_1_i_pokemon_manager}{IPokemon\+Manager} y permite al jugador gestionar sus pokemon, realizar ataques y utilizar ítems.
