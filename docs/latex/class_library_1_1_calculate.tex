\doxysubsection{Descripción detallada}
Clase que implementa la lógica para calcular el daño en un combate entre pokemon. 

Esta clase utiliza la interfaz \doxylink{interface_library_1_1_i_calculate}{ICalculate} para calcular el daño que un pokemon atacante inflige a un pokemon defensor utilizando un movimiento específico.
